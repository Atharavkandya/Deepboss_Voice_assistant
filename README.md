# Deepboss_Voice_assistant
Deepboss is a Python-based voice assistant that can recognize speech commands and perform tasks like opening websites, playing music, and responding with voice feedback. It supports continuous listening mode and provides a simple interactive AI assistant experience.


Deepboss is a voice-controlled virtual assistant built using Python. It can understand voice commands and perform tasks like opening websites, playing music, and responding with voice output.

## 🚀 Features
- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Opens websites (Google, YouTube, etc.)
- 🎵 Plays songs using custom music library
- 🔁 Continuous listening mode
- ⚡ Fast and interactive responses

## 🧠 How it Works
- Uses speech recognition to convert voice to text
- Processes commands using Python logic
- Responds using text-to-speech engine
- Executes tasks like opening browser or playing music

## 🛠️ Tech Stack
- Python
- SpeechRecognition
- pyttsx3 / gTTS
- Webbrowser module
