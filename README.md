# Deepboss_Voice_assistant
Deepboss is a Python-based voice assistant that can recognize speech commands and perform tasks like opening websites, playing music, and responding with voice feedback. It supports continuous listening mode and provides a simple interactive AI assistant experience.


Deepboss is a voice-controlled virtual assistant built using Python. It can understand voice commands and perform tasks like opening websites, playing music, and responding with voice output.

## 🚀 Features
- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Opens websites (Google, YouTube, etc.)
- 🎵 Plays songs using custom music library
- 🔁 Continuous listening mode
- ⚡ Fast and interactive responses

## 🧠 How it Works
- Uses speech recognition to convert voice to text
- Processes commands using Python logic
- Responds using text-to-speech engine
- Executes tasks like opening browser or playing music

## 🛠️ Tech Stack
- Python
- SpeechRecognition
- pyttsx3 / gTTS
- Webbrowser module

# 🤖 Deepboss AI Assistant

A voice-controlled desktop assistant built using Python that can open applications, control websites, perform login actions, and execute voice-based searches.

---

## 🚀 Features

### 🔹 Version 1 (v1.0)
- 🎤 Voice recognition
- 🖥️ GUI interface (Tkinter)

#### 📱 App Control:
- Open Chrome
- Open Notepad
- Open VS Code
- Open Calculator

#### 🌐 Website Control:
- Open Google
- Open YouTube
- Open LinkedIn
- Open Facebook
- Open Instagram

#### 🔐 Voice-based Login (Safe Mode):
- Opens login pages
- Takes username & password via voice
- Automatically types credentials
- ❗ No data is stored (secure approach)

---

### 🔹 Version 2 (v2.0)
- 🔎 Voice-based Google search  
  *(e.g., "search google internships")*

- 🤖 Voice-based ChatGPT search  
  *(e.g., "search chatgpt python roadmap")*

- ⚡ Faster and smarter command handling
